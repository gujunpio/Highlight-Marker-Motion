# Highlight-Marker-Motion

A Chrome Extension that allows you to create customizable, smooth highlight motion animations line-by-line across any webpage text. Perfect for screen recording video tutorials without needing post-production motion graphics.

### Features
- **Custom Highlights & Text Colors**
- **Thickness Control**: Thin, Medium, Thick
- **Animation Style**: Sequential or Overlapping
- **Speed Control**: Slow, Medium, Fast
- **Dark/Light Mode UI**
- **Pop-out Persistent Window**

### Shortcuts
- `Ctrl+Shift+U`: Play / Replay Animation
- `Ctrl+Shift+Z`: Clear Animation
